# UDB-ArbolBST-2026

Visualizador de **Árbol Binario de Búsqueda (BST)** — C# · .NET Framework 4.7.2 · Windows Forms.

## Requisitos

- Visual Studio 2022 (o 2019) con carga de trabajo **Desarrollo de escritorio con .NET**

## Cómo abrir y compilar

1. Clonar o descomprimir el repositorio.
2. Abrir `ArbolBST.sln`.
3. Compilar: **Ctrl+Shift+B**
4. Ejecutar: **F5**

## Estructura

- `Models/NodoBST.cs` — nodo del árbol
- `Services/ArbolBST.cs` — insertar, buscar (con camino), InOrden, limpiar
- `Forms/FormPrincipal.cs` — interfaz (completada por el equipo de frontend)

## Ramas Git

- `main` — versión estable
- `develop` — integración
